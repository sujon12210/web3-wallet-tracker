# Web3 Wallet Tracker

This project helps you track a crypto wallet.
You can check balance and recent transfers.
It uses public blockchain APIs and Web3 concepts.

This project is made for beginners.
All code is simple and easy to read.

## Features
- Check wallet balance
- View recent transactions
- Support for Ethereum testnet
- Clean and simple structure

## Requirements
- Node.js installed
- Internet connection
- A free API key from a blockchain explorer

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add your API key inside .env
6. Run: node index.js

## Example
Enter a wallet address when asked.
The app will show balance and last transactions.

## Learning Goals
- Understand how Web3 works
- Learn how to call blockchain APIs
- Learn basic crypto wallet tracking
